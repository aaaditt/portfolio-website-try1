# Aadit Chandra | Interactive Liquid Portfolio

A premium, interactive personal portfolio website featuring liquid distortion effects, built with HTML, Tailwind CSS, and embedded custom animations.

## Features

- **Liquid Physics:** Interactive ripple and displacement filter effects on hover and click.
- **Glassmorphism Design:** Semitransparent panels with background blurs.
- **Dark/Light Mode:** Seamless theme toggling with persistent preferences.
- **Responsive Layout:** A clean masonry grid for projects and selected works.
- **Animations:** Custom float, pulse, and text distortion animations built with CSS keyframes and SVG filters.

## Local Development

1. Clone the repository.
2. Open `index.html` directly in your browser or serve it using a local development server:

   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (serve)
   npx serve .
   ```

3. Navigate to `http://localhost:8000` or the port provided by your server.

## Structure

- `index.html`: Main landing page showcasing the hero section, journey, about, and selected works.
- `contact.html`: Dedicated contact page with cinematic zoom animations and interactive forms.
